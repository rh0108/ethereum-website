## Contract Addresses

Addresses of deployed AZTEC contracts on Ethereum mainnet and testnets.

## Usage

Install the module:

```bash
$ yarn add @aztec/contract-addresses
```

And import it in your project:

```js
const aztecContractAddresses = require('@aztec/contract-addresses');
```
