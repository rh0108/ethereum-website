## Dev utils

Dev utils to be shared across AZTEC projects and packages.

## Usage

Install the module:

```bash
$ npm install @aztec/dev-utils --global
```

And import it in your project:

```js
const aztecDevUtils = require('@aztec/dev-utils');
```
