## bn128

Barreto-Naehrig pairing friendly curve providing 128 bits of security defined as: y2=x3+b.

## Usage

Install the module:

```bash
$ yarn add @aztec/bn128
```

And import it in your project:

```js
const bn128 = require('@aztec/bn128');
```
