# Contract Artifacts

Smart contract compilation artifacts for the latest version of AZTEC.

## Usage

Install the module:

```bash
$ yarn add @aztec/contract-artifacts
```

And import it in your project:

```js
const aztecContractArtifacts = require('@aztec/contract-artifacts');
```
